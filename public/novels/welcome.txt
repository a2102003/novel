第1章 欢迎使用 ZenReader

这是一个完全运行在本地的静态阅读器，但支持通过配置文件预加载小说。

如果您希望其他人也能看到某本小说，请按以下步骤操作：

1. 将小说的 .txt 文件放入项目代码的 `public/novels/` 文件夹中。
2. 打开 `public/novels/manifest.json` 文件。
3. 在列表中添加新的书籍信息，格式如下：

{
  "title": "您的书名",
  "file": "文件名.txt"
}

例如：
[
  {
    "title": "使用指南",
    "file": "welcome.txt"
  },
  {
    "title": "西游记",
    "file": "xiyouji.txt"
  }
]

4. 重新部署网站，所有访问者就都能看到这些书了。

第2章 功能介绍

- 极致的阅读体验：支持日间、夜间、羊皮纸三种模式。
- 智能解析：自动识别章节标题。
- 进度保存：自动记录您的阅读进度（保存在您自己的浏览器中）。
- AI 助手：集成了 Google Gemini，可以帮您总结章节或回答剧情问题。
